# Task Management System - Frontend

A modern **React 19+ frontend** built with **Vite, TypeScript, Tailwind CSS, and Redux Toolkit**.  
This serves as the UI for the Task Management System.

### Installation

1. Navigate to the frontend folder:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Create a `.env` file:
   ```env
   VITE_API_URL=http://localhost:7134
   ```

4. Start the dev server:
   ```bash
   npm run dev
   ```

5. Open in browser:
   ```
   http://localhost:5173
   ```
